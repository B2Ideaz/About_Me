Augmented Reality for Scientific Experiments

AR technology enables immersive scientific experiments without real-world hazards. Biology students can perform virtual dissections, Chemistry students can mix chemicals safely, and Physics students can simulate zero-gravity experiments.

Halophiles for Desalination

Using salt-eating bacteria to desalinate seawater efficiently and reduce waste brine, potentially lowering desalination costs.

Sports Facility Mapping App

An app that maps all public sports facilities using Google Maps API and on-the-ground agents. Revenue can be generated through ads.

AI-Powered Background Noise Removal

An extension that removes background noise from video calls using AI, allowing meetings in noisy environments.
